Mini Portfolio

A compact dark card with profile image, animated typed role, bio, social icons, and a hire button. No dependencies.

akhersheikh.github.io/free-template/preview/minimal-portfolio/
What's included
  • Card layout — standalone page or embeddable component
  • Typewriter animation for role — pure JS, no library
  • Profile image with warm border accent
  • Social icon buttons — easily extended
  • Responsive — stacks on mobile
  • Popup contact link
  • Zero dependencies — plain HTML, CSS & JS only
How to use

Download the ZIP or clone the repository. Open mini-portfolio.html in your browser and start editing. No build step, no npm install.

All editable sections are marked with a ✏️ comment inside the file.

bash
git clone https://github.com/AkherSheikh/free-template.git cd free-template/mini-portfolio open mini-portfolio.html
